Detailed Description

Platform, compiler, attribute and type definitions.

This file is meant to be included from virtually every header file of the base library. It contains macros for checking what the targeted platform is, what compiler is being used, what relevant platform and compiler features are available, and so on. Finally, it performs some tests to ensure that the targeted platform and compiler are supported by the library.
