ip.h File Reference

Detailed Description

Internet Protocol (IP) utilities.

The Internet Protocol serves as foundation for several other important networking protocols, such as TCP (see tcp.h) and UDP (see udp.h). Extending protocols use the IP addressing schema when specifying message recipients.

This file provides structures, constants and functions useful for representing and checking IP addresses.

ah_extern ah_err_t ah_ipaddr_v4_stringify ( const struct ah_ipaddr_v4 *  addr,
char *  dest,
size_t *  dest_size 
)

Writes dot-decimal representation of addr and a NULL terminator to dest.

If the representation does not fit in dest, dest is left unmodified.

Parameters
addrPointer to IPv4 address.
destPointer to buffer to receive stringification.
dest_sizeSize of dest, in bytes.
Returns
One of the following error codes:
  • AH_ENONE - If the complete representation could be written to dest.
  • AH_EINVAL - If any of addr, dest or dest_size is NULL.
  • AH_EOVERFLOW - If the representation would not fit in dest.
See also
https://datatracker.ietf.org/doc/html/rfc1123#section-2.1
