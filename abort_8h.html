abort.h File Reference

Detailed Description

Process abortion routines.

Various functions for abnormally terminating the application process, typically to indicate that some irrecoverable behavior has been observed.

Note
In contrast to the abort() function of the C99 standard library, the functions here must be able to guarantee what the results of calling them are.

ah_extern ah_noreturn void ah_abort ( void  )

Aborts application process without a message.

Invoking this function results in the following reactions on each supported platform:

  • [Darwin, Linux] The SIGABRT signal is unblocked and then raised.
  • [Win32] The SIGABRT signal is raised.
