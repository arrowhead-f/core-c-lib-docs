ah_http_chunk Struct Reference

Detailed Description

HTTP chunk, including data and an arbitrary extension.

Used with ah_http_client_send_chunk() to send chunked HTTP messages.

Note
Some fields of this data structure are private in the sense that a user of this API should not access them directly. All private fields have names beginning with an underscore.

const char* ah_http_chunk::ext

Arbitrary chunk extension.

This field must be NULL, contain an empty NULL-terminated string, or adhere to the chunk-ext syntax outlined in RFC9112.
