ah Directory Reference

Detailed Description

Arrowhead header folder.

As a matter of organization, all official Arrowhead libraries keep their headers in a folder named ah directly inside an include folder. This makes it possible to include any of its headers using a C statement such as #include <ah/loop.h>.
