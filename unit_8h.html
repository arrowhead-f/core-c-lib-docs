unit.h File Reference

Detailed Description

Unit testing utilities.

This file provides data structures and functions for writing automated unit tests. To use these utilities, you must first allocate an ah_unit_res instance and zero its memory. You then pass it to the assertion functions provided here, such as ah_unit_assert() or ah_unit_assert_eq_uintmax(). When all relevant assertions have been executed, you call ah_unit_print_results() with your result accumulator, which prints a summary of your assertion results.

You are free to use the functions and structures here however you wish, and organize them in the manner best suited for the software you wish to test.

Note
On platforms where a distinction is made between regular printing and error printing (such as STDOUT and STDERR on POSIX systems), failure messages are printed to the latter.

The comparison is made using C99 memcmp() after a check that actual_size is equal to expected_size. The only difference between this function and ah_unit_assert_eq_mem() is that this function assumes actual and expected to contain printable characters.
