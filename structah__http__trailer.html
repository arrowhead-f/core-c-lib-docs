ah_http_trailer Struct Reference

Detailed Description

HTTP chunked message end, consisting of a last chunk extension and a trailer.

An instance of this message is used to end a chunked HTTP transmission in a call to ah_http_client_send_trailer(). It allows for you to specify an extension for the last-chunk and any headers for the trailer-section.

Note
Some fields of this data structure are private in the sense that a user of this API should not access them directly. All private fields have names beginning with an underscore.

Arbitrary chunk extension.
