sock.h File Reference

Detailed Description

Socket utilities.

A "socket" is a handle representing the ability to communicate over some supported medium. The term is taken from the BSD operating system family, which popularized the concept and the API that came with it. Every communication facility provided by the base library conceptually provides a form of sockets. This file provides functions and other utilities primarily for dealing with socket addresses, which are used to indicate where messages sent or received via a socket is destined or is expected to come from.

Writes human-readable representation of sockaddr to dest.

If the representation does not fit in dest, dest is left unmodified.

Parameters
sockaddrPointer to socket address.
destPointer to buffer to receive stringification.
dest_sizeSize of dest, in bytes.
Returns
One of the following error codes:
  • AH_ENONE - If the complete representation could be written to dest.
  • AH_EINVAL - If any of sockaddr, dest or dest_size is NULL.
  • AH_EOVERFLOW - If the representation would not fit in dest.
