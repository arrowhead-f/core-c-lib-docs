rw.h File Reference

Detailed Description

Memory reader/writer.

This file provides ah_rw, or the Reader/Writer (R/W), which is a set of pointers that points into a block of memory as follows:

r w e
| | |
V V V
+---+---+---+---+---+---+---+---+---+---+---+---+
Memory block | 1 | 7 | 3 | 2 | 4 | 1 | 0 | 0 | 0 | 0 | 0 | 0 |
+---+---+---+---+---+---+---+---+---+---+---+---+
:.........: :.....................:
: :
Readable bytes Writable bytes

The following invariants must always be satisfied for every initialized reader/writer:

  1. The r pointer must be less than or equal to w.
  2. The w pointer must be less than or equal to e.
  3. All of r, w and e must point into or to the first byte right after the same block of memory.

The most straightforward way to ensure that the invariants remain true for a given ah_rw instance is to initialize and update it only through the functions listed in this header.
