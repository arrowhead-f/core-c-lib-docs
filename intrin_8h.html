Detailed Description

Compiler intrinsics.

A compiler intrinsic is a special function provided by the compiler, causing it to optimize or add to its output. This file provides two categories of intrinsics, (1) such supported by all supported compilers and (2) those only supported by GCC-compatible compilers. The former category are prefixed with the usual ah_, while the latter category have the slightly longer ah_gcc_ prefix.

[Clang, GCC] Adds a and b, storing the sum to result.

Conceptually, a and b are both turned into infinite precision integer types before being added together. The function returns true if the result of the addition is outside the range representable by the type of the variable pointed at by result.

Parameters
aValue of an arbitrary integer type.
bValue of an arbitrary integer type.
resultPointer to value of an arbitrary integer type.
Returns
true if the sum of a and b cannot be represented by result. false otherwise.
