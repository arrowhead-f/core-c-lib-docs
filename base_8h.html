Detailed Description

Compile-time metadata.

This file provides details determined when this library is compiled.

Warning
If you compile this library in any other way than the one officially supported, the macros defined here may present arbitrary default values when used. If the used compiler does not support __has_include, a generated file may be missing, causing compilation to fail unless you provide a substitute.
