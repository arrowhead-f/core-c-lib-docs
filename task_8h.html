task.h File Reference

Detailed Description

Task scheduling.

A task is a function that is to be executed after a certain baseline with a certain context, concretely represented by a user data pointer.

◆ ah_task_schedule_at()

ah_extern ah_err_t ah_task_schedule_at ( ah_task_t *  task,
ah_time_t  baseline 
)

Schedules task for execution at or after baseline.

Parameters
taskPointer to task.
baselineTarget baseline.
Returns
One of the following error codes:
  • AH_ENONE - task scheduled.
  • AH_EDOM [Darwin] - baseline is too far into the future to be acceptable to the platform event queue.
  • AH_EINVAL - task is NULL.
  • AH_ENOBUFS [Darwin, Linux] - task event loop has no more slots available in its event queue and could not flush it to make more slots available.
  • AH_ENOMEM [Darwin, Linux, Win32] - task event loop failed to allocate heap memory.
  • AH_ESTATE - task is already scheduled and has not yet been cancelled or executed.
Warning
If the ah_loop provided to task is not given opportunity to execute at or after baseline, as reported by ah_time_now(), task will never be executed. Note, however, that task will be invoked with AH_ECANCELED if the event loop is terminated. For task to never be executed, the event loop must never be given enough time to execute after baseline and never be terminated.
